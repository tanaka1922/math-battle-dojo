# Codex 5.3 向け品質チェック指示書

## 対象ファイル
`problems.json`（全2035問）

## チェック内容
`problems.json` の全2035問について、以下の3点をチェックしてください。

### 1. 計算ミスチェック
- 四則演算の答え（`a` フィールド）が正しいか
- 方程式の解を代入して等式が成立するか確認
- 一次関数の代入値が正しいか
- 確率の分子・分母が正しいか
- 面積・体積の計算が正しいか（π=3.14 使用）

### 2. 日本語ミスチェック
- 問題文（`q`）に誤字・脱字がないか
- 解説（`exp`）の説明が正確か
- ヒント（`h1`, `h2`）が問題と整合しているか
- 単位（cm², cm³, °, 通り等）が適切に記載されているか

### 3. 難易度チェック
- 各カテゴリ内で難易度が適切に分散されているか
- `gr` フィールド（中1/中2）と問題の難易度が一致するか
- 同一カテゴリ内に重複・類似しすぎる問題がないか

## 出力形式
修正が必要な問題を `codex-errors.json` に以下の形式で出力してください：

```json
[
  {
    "id": 問題ID,
    "cat": "カテゴリ名",
    "field": "修正が必要なフィールド名（a/q/h1/h2/exp）",
    "current": "現在の値",
    "suggested": "修正案",
    "reason": "修正理由"
  }
]
```

## 優先順位
1. **計算ミス**（最優先 - 答えが間違っているもの）
2. **日本語ミス**（次点 - 意味が通じないもの）
3. **難易度の偏り**（参考情報として）
